Foundation

The Foundation category of HL7® FHIR® resources includes Conformance, Terminology, Security, Documents, and Other.

Conformance

Resources in the Conformance category are used to represent metadata about the datatypes, resources, and API features of the FHIR specification and can be used to create derived specifications.

Click here to see the detailed categorization and full set of resources that fall into this category in the HL7® FHIR® Standard.

Implemented Resources

  • Capability Statement
  • Structure Definition
  • Operation Definition

Security

The Security resources track the trustworthiness and reliability of a resource and a consumer's choice to permit or deny access to their healthcare records.

Implemented Resources

  • Provenance
  • Consent

Documents

Documents are used to search, locate, relate, and write documents belonging to a patient. This includes clinical notes like History and Physical, Continuity of Care Documents (CCD), and diagnostic reports such a Radiology and Pathology reports.

Implemented Resources

  • DocumentReference

Other

Resources in the Other category include uncategorized Foundation resources.
